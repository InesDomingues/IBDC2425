O que são dados clínicos?

  • Definição: Dados clínicos referem-se a informações recolhidas de várias fontes, como pacientes, dispositivos médicos e sistemas de saúde, utilizadas para fins de diagnóstico, tratamento e investigação.
  • Dados Clínicos Relacionados com Pacientes: Incluem histórico médico, dados demográficos, resultados de laboratório, estudos de imagem, sinais vitais e avaliações clínicas.
  • Dados Clínicos Gerados por Dispositivos Médicos: Gerados por dispositivos como ECGs, monitores de pressão arterial ou dispositivos de fitness usados no corpo.
  • Dados Clínicos Gerados pelo Sistema de Saúde: Incluem registos de saúde eletrónicos (EHRs), sistemas de informação hospitalar e sistemas de informação laboratorial.

A importância dos dados clínicos

  • Investigação e Desenvolvimento: Base para estudos de pesquisa e ensaios clínicos.
  • Medicina Baseada em Evidências: Fornecem evidências empíricas para protocolos de tratamento e diretrizes de cuidados de saúde.
  • Cuidados e Monitorização do Paciente: Auxiliam na monitorização da saúde dos pacientes e na avaliação das respostas aos tratamentos.
  • Desenvolvimento de Dispositivos Médicos: Fundamentais no desenvolvimento e avaliação de dispositivos médicos.
  • Melhoria da Qualidade: Identificam padrões e tendências para melhoria na prestação de cuidados de saúde.
  • Otimização do Sistema de Saúde: Permitem decisões orientadas por dados, otimizando a alocação de recursos.
  • Investigação Colaborativa e Partilha de Dados: Recurso valioso para a colaboração entre investigadores, clínicos e instituições.

Introdução a Bases de Dados

  • Base de Dados: Armazena e manipula dados eletronicamente.
  • Principais Benefícios: Melhor organização de dados, integração de dados, consistência e integridade de dados, consulta e análise eficientes, segurança de dados e privacidade.

Tipos de Bases de Dados

  • Bases de Dados SQL: Relacionais e Analítico (OLAP)
  • Bases de Dados NoSQL: Armazenamento de Documentos, Chave-Valor, Famílias de Colunas, Bases de Dados de Grafos

Bases de Dados Relacionais

  • Tabelas
  • Linhas
  • Colunas

Tabelas

As chaves ligam entidades (tabelas) com base em atributos partilhados

  • Chave Primária: Identificador único para cada registo (linha)
  • Chave Estrangeira: Refere-se à chave primária de outra tabela
  • Chave Composta: Composta por dois ou mais atributos
  • Chave Candidata: Atributo que contém um valor único em cada linha
  • Chave Alternativa: Chave candidata não selecionada como chave primária

Cardinalidade

  • Um-para-Um (1:1)
  • Um-para-Muitos (1:N)
  • Muitos-para-Muitos (N:M)

Aula 8 / 14 - Otimização e Indexação

Conteúdos

  • Boas práticas
  • Otimização de Bases de Dados
  • Indexação
  • B-Trees

Boas práticas

  • Evitar o uso de wildcards no início das pesquisas
  • Preferir INNER JOIN ao OUTER JOIN
  • Evitar o uso de colunas desnecessárias na cláusula SELECT

Otimização da Base de Dados: Indexação

  • Motivação: Índices são essenciais para a recuperação eficiente de dados
  • Considerações: Índices melhoram o desempenho, mas exigem manutenção regular
  • Tipos de índices: Primário (Agrupado) e Secundário (Não Agrupado)

B-Trees

Índices em MySQL são armazenados em B-Trees para manter uma estrutura ordenada, facilitando a pesquisa de correspondências exatas e intervalos.
